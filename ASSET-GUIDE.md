# Photo & Asset Guide — Caitlin Onie Photography

This is a checklist of every photo and file the site needs. You'll upload all of these through the **admin panel** (no GitHub needed). The admin panel handles renaming + uploading automatically — but if you ever need to know what each file *should* be, this is your reference.

## How to upload via the admin

1. Go to `https://[your-site].netlify.app/admin/`
2. Log in with your email
3. Click any image field → upload from your computer
4. Decap CMS will save it to `assets/images/` and use it on the page
5. Click **Publish** when done

---

## File rules (important!)

- **All lowercase** filenames
- **No spaces** — use hyphens (e.g., `hero-1.jpg`, not `Hero 1.jpg`)
- **No special characters** — no `&`, `'`, `()`, etc.
- **Use `.jpg`** for photos, **`.svg`** for logos, **`.png`** for icons with transparency
- **Compress before uploading**: aim for under 500KB per photo. Use [tinypng.com](https://tinypng.com) or [squoosh.app](https://squoosh.app)

### 🖼 Gallery Photos (each category page)

You can upload **as many as you want** per category — minimum 6, maximum unlimited. Mix orientations (portrait + landscape) for the best editorial flow.

- `portraits-1.jpg` through however many you want
- `products-1.jpg` etc.
- `landscapes-1.jpg` etc.
- `weddings-1.jpg` etc.
- `nature-1.jpg` etc.

**Recommended sizes**: 1600px on the long edge. Keep originals safe — these are display-quality copies.
